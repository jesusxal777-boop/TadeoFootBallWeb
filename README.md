# Tadeo's Football Challenge - Versión Web

Este es un videojuego de fútbol arcade desarrollado en HTML5, CSS3 y JavaScript ES6 para el cumpleaños número 11 de Tadeo.

## Características

- **Juego Arcade 2D** con vista lateral
- **Selección Mexicana** como protagonista
- **Campaña progresiva** contra 5 rivales (Estados Unidos, Brasil, Argentina, Francia, Inglaterra)
- **Mecánica de Super Disparo** con barra de energía
- **IA adaptativa** que aumenta dificultad según el partido
- **Controles para PC y dispositivos táctiles**
- **Compatible con GitHub Pages** (sin backend requerido)

## Controles

### PC (Teclado)
- **Flechas izquierda/derecha**: Moverse
- **Espacio**: Saltar
- **X**: Patear
- **Z**: Super Disparo

### Dispositivos Táctiles
- **Botones virtuales** en la parte inferior de la pantalla
- Interfaz optimizada para jugar con los pulgares

## Cómo Jugar

1. Abre `index.html` en tu navegador
2. Haz clic en "Jugar"
3. Gana los 5 partidos para celebrar el cumpleaños de Tadeo
4. Cada victoria desbloquea el siguiente rival
5. La dificultad aumenta progresivamente

## Instalación para GitHub Pages

1. Copia todos los archivos a tu repositorio de GitHub
2. Ve a **Settings** → **Pages**
3. Selecciona la rama `main` como fuente
4. Tu juego estará disponible en `https://tu-usuario.github.io/tu-repositorio`
